TypeScript

  1. npm init(initialize-package.json)
  2. Install typescript : npm install --save-dev typescript
  3. Add Folder (dir\src) then add a ts file "Message.ts"
  4. Add starting point of application : 'dir\app.ts'
  5. Add tsconfig.json for ts compilation and mention the starting point
  6. compile the ts file with cmd : node_modules\.bin\tsc
  7. We can use task runner to compile and minify all js files Grunt/Gulp/Webpack
  8. Install ts-loader : npm install --save-dev ts-loader
  9. Install webpack : npm install --save-dev webpack
  10. Add Index.html file
  11. Install http server : npm install --save-dev http-server
  12. Run http server : node_modules\.bin\http-server
  13. To debug ts file we can use sourcemap, for that add compileroption to tsconfig.json
  14. To debug ts file add "devtool" into webpack
  15. To Monitor and watch all ts file , we can use "watch"
Task Runners Vs Bundlers
Executes series of js tasks at client side
Transpile -> SASS to CSS
