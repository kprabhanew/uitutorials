Angular 2

What is module in Angular 2 ?

Angular Module is different from JS module.It is a just a way of bringing various parts of angular appication in single unit.Every Angular2 module nothing but class.

Setup Angualr 2

  1. Install angularjs packages
  2. Add Folder 'app'
  3. Add ts file inside app (app\test.ts)
  4. Add tsconfig.json file to Compile ts file
  5. Compile the ts file using tsc command (dir\node_modules\.bin\tsc)
  6. We can compile Typescript using three methods a. cmd, b. VS Build, c.Gulp[1."dir\node_modules\.bin\gulp", 2. Task Runner Option from solution]
  7. Add the Component file
  8. Add the AppModule file (Typescript stuff completed)
  9. Hook all these together and get them into our web page (Systemjs)
  10. Add "systemjs.config.js" file
  11. Add "Index.html" page

Example : systemjs.config.js 

    (function(global) {
    System.config({
        paths: {
            //paths serve aas alias
            'npm:': 'node_modules/'

            //'npm:': 'node_modules/' Can cause error so follow the above syntax
        },



        //map tells the System loader where to look for things
        map: {
            //our app is within the app folder
            app: 'app',
            //angular bundles
            '@angular/core': 'npm:@angular/core/bundles/core.umd.js',
            '@angular/common': 'npm:@angular/common/bundles/common.umd.js',
            '@angular/compiler': 'npm:@angular/compiler/bundles/compiler.umd.js',
            '@angular/platform-browser': 'npm:@angular/platform-browser/bundles/platform-browser.umd.js',
            '@angular/platform-browser-dynamic': 'npm:@angular/platform-browser-dynamic/bundles/platform-browser-dynamic.umd.js',
            '@angular/http': 'npm:@angular/http/bundles/http.umd.js',
            '@angular/router': 'npm:@angular/router/bundles/router.umd.js',
            '@angular/forms': 'npm:@angular/forms/bundles/forms.umd.js',

            //other librarires
            'rxjs': 'npm:rxjs'
        },
        //packages tells the system loader how to load when no filename and /or no extension

        packages: {
            app: {
                main: './main.js',
                defaultExtension: 'js'
            },
            rxjs: {
                defaultExtension: 'js'
            }
        }
    });
})(this)

Example : app/app.module.ts 

    //Importing javascript modules
import {NgModule} from '@angular/core'; 
import {BrowserModule} from '@angular/platform-browser';
import {HelloWorldComponent} from './helloworld.component'

//Importing AngularJS modules
@NgModule({
    imports:[BrowserModule],
    declarations:[HelloWorldComponent],
    bootstrap:[HelloWorldComponent] // Start app with helloworld component
})

//Module Bootstraping
export class AppModule{}
